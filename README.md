# Data Engineering Trainee Assessment

## Project Objective

The objective of this project is to perform basic data engineering tasks on an e-commerce dataset.
This includes setting up a data engineering project structure, designing a data warehouse model, writing SQL queries, performing data transformations, and implementing data quality checks.

The goal is to demonstrate understanding of fundamental data engineering concepts such as data modeling, data transformation, and data validation.

---

## Dataset Description

The dataset contains e-commerce sales transaction data. Each record represents a purchase made by a customer.

The dataset includes the following columns:

| Column         | Description                |
| -------------- | -------------------------- |
| order_id       | Unique order identifier    |
| customer_id    | Unique customer identifier |
| product_name   | Product purchased          |
| category       | Product category           |
| region         | Sales region               |
| quantity       | Number of items purchased  |
| unit_price     | Price per item             |
| revenue        | Total revenue generated    |
| payment_method | Payment method used        |
| order_date     | Date of purchase           |

This dataset is used for performing SQL analysis, data transformation, and data quality checks.
